Detailed Description

Represents an access token which is used to access the LINE Platform.

Most API calls to the LINE Platform require an access token as evidence of successful authorization. A valid access token is issued after the user grants your app the permissions that your app requests. An access token is bound to permissions (scopes) that define the API endpoints that you can access. Choose the permissions for your channel in the LINE Developers site and set them in the login method used in your app.

An access token expires after a certain period. ExpiresIn specifies the time until which this access token expires.

By default, the LINE SDK stores access tokens in a secure place on the device running your app and obtains authorization when you access the LINE Platform through the framework request methods.

Don't try to create an access token yourself. You can get the stored access token with fewer properties with FLineSDK::GetCurrentAccessToken().
